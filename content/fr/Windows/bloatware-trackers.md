## Windows10Debloater

Script pour supprimer les bloatwares de Windows 10. [Download](https://github.com/Sycnex/Windows10Debloater)

```powershell
iwr -useb https://git.io/debloat|iex
```

## O&O ShutUp10

Outil anti-espion gratuit pour Windows 10 et 11. [Download](https://www.oo-software.com/fr/shutup10)

```powershell
iwr -Uri "https://dl5.oo-software.com/files/ooshutup10/OOSU10.exe" -OutFile "OOSU10.exe"
```

## Privatezilla

Privatezilla est le moyen le plus simple d'effectuer une vérification rapide de la confidentialité et de la sécurité de votre copie Windows 10. [Download](https://github.com/builtbybel/privatezilla)
